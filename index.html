Project Summary

What is the weather like as we approach the equator? "It gets hotter..." you may think. In reality, the answer is more complex. This project analyzes how weather changes as we move closer to the equater. To accomplish this analysis, we utilized data from the OpenWeatherMap API to assemble a data set of over 500 randomly-selected cities.

After assembling the data set, we visualized various aspects of the weather versus latitude using Matplotlib. Factors included: temperature, cloudiness, wind speed, and humidity. This site provides the source data and visualization is created as part of the analysis, as well as explanations and descriptions of any trends and correlations witnessed.
